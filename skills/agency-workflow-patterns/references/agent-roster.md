## Agent Selection Decision Tree

```
Need UI work?
├─ Yes → Frontend Developer
└─ No → Continue

Need backend/API work?
├─ Yes → Backend Architect
└─ No → Continue

Need testing?
├─ Yes → Reality Checker (general) or API Tester (APIs) or Performance Benchmarker (performance)
└─ No → Continue

Need planning?
├─ Yes → Senior Project Manager or Plan agent
└─ No → Continue

Need research?
├─ Yes → Explore agent (codebase) or Trend Researcher (market)
└─ No → Continue

Complex/unclear?
└─ Yes → Orchestrator or Senior Developer
```
